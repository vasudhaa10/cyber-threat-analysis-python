# 🛡️ Cyber Threat Analysis using Python

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on a cybersecurity attack dataset using Python. The objective is to analyze attack patterns, protocol usage, severity levels, traffic distribution, and anomaly scores to gain meaningful security insights through data visualization.

---

## 🎯 Project Objective

- Analyze cybersecurity attack data.
- Identify attack trends and patterns.
- Visualize different attack characteristics.
- Understand anomaly score distribution.
- Practice Python-based Exploratory Data Analysis (EDA).

---

## 📊 Dataset Information

- Total Records: **40,000**
- Total Features: **25**
- Dataset Type: Cybersecurity Attack Logs

---

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📈 Analysis Performed

- Data Loading
- Dataset Information
- Missing Value Analysis
- Attack Type Distribution
- Severity Level Analysis
- Protocol Analysis
- Traffic Type Analysis
- Network Segment Analysis
- Log Source Analysis
- Anomaly Score Distribution
- Correlation Heatmap
